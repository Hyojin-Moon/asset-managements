# 품질 검증 체크리스트

다음 단계로 진행하기 전 각 단계에서 반드시 검증해야 할 항목입니다.

## 빌드 & 컴파일

- [ ] `npm run build-dev` 오류 없이 완료
- [ ] TypeScript 타입 에러 없음

## 테스트

- [ ] `npm test` 모든 테스트 통과
- [ ] 새 기능에 대한 테스트 추가 (필요시)

## 코드 품질

- [ ] Import 순서 규칙 준수 (CLAUDE.md 참조)
- [ ] 코드 포맷팅 일관성 유지
- [ ] console.log 및 디버그 코드 제거

## 기능 검증

- [ ] 로컬 환경 (`http://localhost:3001`)에서 수동 테스트 완료
- [ ] 기존 기능에 대한 회귀 없음
- [ ] 엣지 케이스 테스트 완료

## 웹뷰 검증 (해당시)

- [ ] iOS 시뮬레이터에서 테스트
- [ ] Android 에뮬레이터에서 테스트
- [ ] 네이티브 브릿지 함수 동작 확인

## 트래킹 검증 (해당시)

- [ ] Amplitude 이벤트 발송 확인
